# Synthetic Dataset Documentation

## Overview

The FedAgent-Chain synthetic dataset is generated by `scripts/generate_synthetic_data.py`
using the schema and distributions defined in `src/data/synthetic_generator.py`.

All records are **completely fictitious** — no real personal data is included.

## Generation

```bash
python scripts/generate_synthetic_data.py \
    --config configs/experiment/fedagent_chain_full.yaml \
    --seed 42 \
    --output-dir data/synthetic/
```

## Scale (Default — Paper Settings)

| Entity | Count |
|--------|-------|
| Total users | 10,000 (2,500 per node) |
| Total jobs | 5,000 (1,250 per node) |
| Suitability pairs | 50,000 (12,500 per node) |
| Positive label rate | ~45–55% |
| Consent rate | ~92% |

## Schema

### users.csv / per-node `{node_id}/users.csv`

| Column | Type | Description |
|--------|------|-------------|
| `user_id` | str (UUID) | Synthetic identifier — no PII |
| `node_id` | str | Regional node: saudi_arabia / united_states / china / europe |
| `skill_vector` | List[int] (50) | Binary occupational skill vector (O*NET/ESCO crosswalk) |
| `education_level` | int [0-4] | 0=none, 1=secondary, 2=vocational, 3=undergraduate, 4=postgraduate |
| `disability_category` | str | One of: mobility, vision, hearing, cognitive, communication, mental_health, chronic_health, multiple |
| `accommodation_needs` | List[int] (20) | Binary accommodation requirement flags |
| `language_primary` | str | ISO 639-1 code (e.g., ar, en, zh) |
| `language_secondary` | str or null | Optional secondary language |
| `preferred_work_mode` | str | onsite / hybrid / remote |
| `employment_goal` | str | fulltime / parttime / freelance / internship |
| `consent_given` | bool | **Must be True for record to be included in training** |

### jobs.csv / per-node `{node_id}/jobs.csv`

| Column | Type | Description |
|--------|------|-------------|
| `job_id` | str (UUID) | Synthetic job identifier |
| `node_id` | str | Regional node |
| `required_skills` | List[int] (50) | Binary required skill vector |
| `accessibility_score` | float [0,1] | Job's disability-accessibility rating |
| `work_mode` | str | onsite / hybrid / remote |
| `language_required` | str | ISO 639-1 code |
| `education_minimum` | int [0-4] | Minimum education level |
| `accommodation_provided` | List[int] (20) | Binary accommodation provision flags |
| `sector` | str | Industry sector |

### outcomes.csv / per-node `{node_id}/outcomes.csv`

| Column | Type | Description |
|--------|------|-------------|
| `user_id` | str | Reference to users.csv |
| `job_id` | str | Reference to jobs.csv |
| `node_id` | str | Regional node |
| `suitability_label` | int [0,1] | Binary match label |
| `suitability_score` | float [0,1] | Continuous suitability score |
